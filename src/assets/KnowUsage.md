# 船舶故障知识库导入说明

系统知识库面向船舶装备故障诊断场景，适合导入以下内容：

- 主推进发动机、辅机、液压系统、电气设备、甲板机械等装备的故障案例
- 维修手册片段、巡检规范、操作规程、注意事项
- 故障现象、可能原因、维修步骤、风险提示等结构化文本

## 支持格式

- `.txt`：按段落切分导入
- `.jsonl`：每行一个知识对象
- `.pdf`：自动抽取文本后分段导入

## JSONL 推荐结构

每行一个 JSON 对象，建议至少包含以下字段：

```json
{"name":"主机启动困难","equipment":"主推进发动机","category":"故障案例","desc":"主机冷启动耗时长，伴随排烟发白。可能原因包括燃油压力不足、喷油器雾化不良、电瓶电压低。建议先检查启动空气压力、燃油滤器和喷油正时。"}
{"name":"液压泵异响","equipment":"液压系统","category":"维修手册","desc":"液压泵在高负载工况下出现金属摩擦声，应重点检查吸油口进气、轴承磨损和油液污染情况。"}
```

## 字段建议

- `name`：知识名称，建议直接写故障名称或手册章节标题
- `equipment`：所属装备类型，如“主推进发动机”“液压系统”“电气设备”
- `category`：可选，建议使用“故障案例”“维修手册”“操作规程”“注意事项”等分类
- `desc`：正文内容，建议包含症状、原因、排查步骤和处理建议

## 编写建议

- 一条知识聚焦一个故障或一个维修主题，避免内容过散
- `desc` 尽量写完整句，至少包含可检索的专业术语
- 同一知识不要重复导入，便于后续索引构建和检索
- 批量导入、删除后建议执行一次“重建索引”
